# 프로젝트 명
[졸업 프로젝트] 공공데이터 활용 대중교통 알람 어플

## 개요
특정 버스나 지하철의 기본정보 및 도착정보를 알려주고 도착 시간에 가까워지면 알람 메시지를 울리게 하는 어플리케이션
